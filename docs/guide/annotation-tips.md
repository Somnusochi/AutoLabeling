# 标注最佳实践

高质量的标注数据是训练出优秀模型的关键。本指南分享 VLM 预标注和人工修正的经验技巧。

## VLM 预标注技巧

### 1. 类别描述要具体

**好的描述**：
```
red car, blue truck, person wearing helmet
white cat, black dog, brown bird
laptop computer, mobile phone, tablet device
```

**差的描述**：
```
vehicle, animal, device
object, thing, item
```

**为什么**：VLM 对具体描述的理解更准确，能生成更精准的边界框。

### 2. 使用英文效果更好

虽然系统支持中文，但英文描述的检测效果通常更好：

```
# 推荐
person, car, bicycle

# 可用但效果略差
人, 汽车, 自行车
```

### 3. 分批次检测

对于复杂场景，分多次检测不同类别：

```
第一次：person, car
第二次：traffic light, stop sign
第三次：bicycle, motorcycle
```

**优势**：
- 避免类别过多导致检测质量下降
- 可以针对不同类别调整过滤策略
- 更容易发现问题并修正

### 4. 利用自然语言描述

LocateAnything-3B 支持自然语言，可以描述更复杂的场景：

```
person holding umbrella
car with open door
dog running on grass
bottle on table
```

### 5. 图片质量很重要

- **清晰度**：模糊的图片检测效果差
- **光照**：过暗或过曝会影响检测
- **角度**：极端角度的目标可能检测不到
- **遮挡**：严重遮挡的目标检测困难

**建议**：优先标注质量好的图片，质量差的图片可以跳过或手动标注。

## 人工修正技巧

### 1. 先整体后局部

**推荐流程**：
1. 先用「全部」模式查看 VLM 的完整检测结果
2. 切换到「最优」模式，检查每个类别的最佳框
3. 使用「NMS」模式去除重叠框
4. 最后进入「标注」模式精细调整

### 2. 合理使用过滤模式

| 模式 | 适用场景 | 说明 |
|------|---------|------|
| **全部** | 初步检查 | 查看所有检测框，了解 VLM 的检测情况 |
| **最优** | 快速筛选 | 每个类别只保留一个最佳框，适合类别明确的场景 |
| **NMS** | 去重 | 去除重叠框，IoU 阈值可调（0.3-0.7） |

**NMS IoU 调参建议**：
- **0.3-0.4**：严格去重，适合目标密集场景
- **0.5-0.6**：平衡去重，通用推荐
- **0.7-0.8**：宽松去重，保留更多框

### 3. 标注模式的技巧

#### 快速添加标注框
1. 从历史记录选择类别（点击类别标签）
2. 在画布上拖拽绘制
3. 松开鼠标自动保存

#### 精确调整框位置
- 点击框的边缘可以微调位置
- 使用键盘方向键精确移动
- 按住 Shift 键可以等比例缩放

#### 批量删除
- 框选多个框后按 Delete
- 或使用「全部删除」清空当前图片的所有框

### 4. 保存过滤设置

如果某个过滤策略效果好，点击「保存过滤结果」：
- 设置会持久化到数据库
- 导出和训练时会自动应用
- 避免每次重复调整

## 视频标注技巧

### 1. 选择合适的提取方式

| 方式 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| **场景切换** | 画面变化大的视频 | 自动检测关键帧 | 可能遗漏相似场景 |
| **运动检测** | 目标移动明显的视频 | 捕捉运动目标 | 对静止目标效果差 |
| **固定间隔** | 需要均匀采样的视频 | 简单可控 | 可能提取冗余帧 |

**推荐**：先用「场景切换」，如果效果不好再尝试其他方式。

### 2. SSIM 去重调参

SSIM 阈值控制去重程度（0.5-1.0）：
- **0.90-0.95**：严格去重，大幅减少帧数
- **0.95-0.98**：平衡去重，通用推荐
- **0.98-1.0**：宽松去重，保留更多帧

**建议**：先用默认值 0.95，根据结果调整。

### 3. 关键帧选择策略

**不要全选**：
- 视频帧数通常很多（几百到几千）
- 全选会导致标注工作量巨大
- 很多帧内容相似，标注价值低

**推荐做法**：
1. 浏览时间轴，选择有代表性的帧
2. 优先选择目标清晰、角度多样的帧
3. 一般 50-200 帧足够训练

## 数据集准备技巧

### 1. 数据量建议

| 任务复杂度 | 推荐图片数 | 说明 |
|-----------|-----------|------|
| **简单**（1-2 个类别） | 100-300 张 | 如检测人、车 |
| **中等**（3-5 个类别） | 300-500 张 | 如交通场景 |
| **复杂**（6+ 个类别） | 500-1000 张 | 如复杂工业场景 |

**注意**：质量比数量更重要。100 张高质量标注 > 1000 张低质量标注。

### 2. 类别平衡

确保每个类别的样本数量相对均衡：

**好的分布**：
```
person: 150 张
car: 120 张
bicycle: 100 张
```

**差的分布**：
```
person: 500 张
car: 50 张
bicycle: 10 张
```

**解决方法**：
- 增加少数类别的样本
- 减少多数类别的样本
- 使用数据增强（旋转、翻转、裁剪）

### 3. 数据集划分

推荐的训练/验证/测试比例：

| 数据集大小 | Train | Val | Test | 说明 |
|-----------|-------|-----|------|------|
| **< 200 张** | 70% | 20% | 10% | 小数据集，多留验证集 |
| **200-500 张** | 80% | 10% | 10% | 中等数据集，标准划分 |
| **> 500 张** | 85% | 10% | 5% | 大数据集，多留训练集 |

**系统默认**：70/20/10，适合大多数场景。

### 4. 标注质量检查

导出前检查：
- [ ] 所有目标都被标注
- [ ] 边界框紧贴目标边缘
- [ ] 类别名称正确且一致
- [ ] 没有遗漏的重要目标
- [ ] 没有错误的标注框

**常见错误**：
- 框太大或太小
- 类别名称拼写错误（如 `preson` vs `person`）
- 同一目标有多个框
- 遮挡目标没有标注

## 高级技巧

### 1. 迭代标注

**流程**：
1. 先用 VLM 标注少量图片（50-100 张）
2. 训练一个初步模型
3. 用这个模型预标注更多图片
4. 人工修正后再次训练
5. 重复直到满意

**优势**：逐步提高模型质量，减少人工工作量。

### 2. 困难样本优先

优先标注这些图片：
- 目标小的图片
- 目标密集的图片
- 光照不好的图片
- 遮挡严重的图片

**原因**：这些是模型最容易出错的场景，标注质量对模型性能影响最大。

### 3. 多轮验证

训练后不要只看 mAP 指标：
1. 在训练集上验证（检查过拟合）
2. 在验证集上验证（检查泛化能力）
3. 在测试集上验证（最终评估）
4. 在真实场景视频上验证（实际应用效果）

**如果训练集 mAP 高但测试集低**：说明过拟合，需要更多数据或数据增强。

## 总结

高质量标注的关键：
1. **VLM 预标注**：用具体的英文描述，分批检测
2. **人工修正**：先整体后局部，合理使用过滤模式
3. **视频标注**：选择合适的提取方式，不要全选
4. **数据集准备**：数量够、类别平衡、质量高
5. **迭代优化**：标注 → 训练 → 验证 → 改进
