# 快速开始

本指南将带你从零开始完成第一次目标检测标注和训练。

## 1. 启动项目

```bash
# macOS / Linux
./start.sh

# Windows
start.bat
```

启动后访问：
- **前端界面**：http://localhost:5173
- **API 文档**：http://localhost:8000/docs

## 2. 上传图片或视频

### 图片模式
1. 在左侧侧边栏选择「图片」标签
2. 点击上传区域，选择一张或多张图片（支持拖拽文件夹）
3. 图片会自动压缩到 1280px 以节省显存

### 视频模式
1. 选择「视频」标签
2. 上传视频文件（支持 MP4/MOV/AVI/MKV）
3. 选择关键帧提取方式：
   - **场景切换**：自动检测画面变化
   - **运动检测**：基于光流法检测运动区域
   - **固定间隔**：按时间间隔提取帧
4. 勾选需要的关键帧，点击「加载到标注队列」

## 3. VLM 自动标注

1. 在「目标类别」输入框中输入要检测的目标类别（英文效果更好）
   - 示例：`person, car, dog`
   - 支持自然语言描述：`red car`, `person wearing hat`
2. 点击「开始检测」
3. VLM 会自动生成边界框标注
   - 第一张结果立即可见
   - 批量上传时后续结果实时追加

**提示**：
- 类别名称越具体，检测效果越好
- 避免过于宽泛的描述（如 `object`）
- 可以使用逗号分隔多个类别

## 4. 人工修正

VLM 标注可能不完美，需要人工调整：

### 查看模式
- **全部**：显示所有检测框
- **最优**：每个类别只保留置信度最高的框
- **NMS**：非极大值抑制，去除重叠框

### 标注模式
1. 切换到「标注」标签
2. 在画布上拖拽绘制新的边界框
3. 输入类别名称（可从历史记录快速选择）
4. 点击不需要的框，按 Delete 删除

### 过滤设置
- 调整 NMS IoU 阈值控制重叠框过滤程度
- 点击「保存过滤结果」将设置持久化

## 5. 导出数据集

### 单图导出
- 在结果页面点击「导出 YOLO (.txt)」
- 直接下载单张图的标注文件

### 批量导出
1. 在历史列表中勾选多张图片
2. 点击「导出全部 (X 张)」
3. 下载包含图片和标注的 ZIP 包

**导出格式**：
```
dataset/
├── images/
│   ├── img1.jpg
│   └── img2.jpg
├── labels/
│   ├── img1.txt
│   └── img2.txt
└── data.yaml
```

## 6. 训练 YOLO 模型

1. 在「YOLO 训练」面板选择要训练的检测记录
2. 选择 YOLO 系列：
   - **YOLOv5**：经典稳定，兼容性好
   - **YOLOv8**：平衡速度与精度
   - **YOLOv11**：最新架构，性能最佳
   - **YOLOv26**：实验性，最前沿
3. 选择模型规格：
   - **n** (nano)：最快，适合边缘设备
   - **s** (small)：速度与精度平衡
   - **m** (medium)：通用推荐
   - **l** (large)：高精度，较慢
   - **x** (xlarge)：最高精度，最慢
4. 设置训练参数：
   - **Epochs**：训练轮数（推荐 100-300）
   - **ImgSz**：输入图像尺寸（推荐 640）
   - **Batch**：批大小（根据显存调整，8-16）
5. 点击「开始训练」
6. 实时查看训练进度：
   - Epoch 进度条
   - Loss 曲线
   - mAP 指标

## 7. 验证模型

### 图片验证
1. 训练完成后，在训练记录中点击「验证」
2. 上传测试图片
3. 调整 Conf 和 IoU 阈值
4. 查看检测结果和置信度

### 视频验证
1. 选择「视频」标签
2. 上传测试视频
3. 选择已训练的模型
4. 点击「验证视频（实时）」
5. 观看实时检测流，可暂停/重播

## 8. 下载模型

训练完成后可以下载：
- **PT 模型**：PyTorch 格式，用于继续训练或推理
- **ONNX 模型**：跨平台部署格式
- **数据集**：包含图片和标注的完整数据集

## 常见问题

**Q: VLM 检测不到目标？**
A: 尝试更具体的类别描述，或检查图片质量

**Q: 训练时显存不足？**
A: 降低 Batch Size 或选择更小的模型规格（如 n 或 s）

**Q: 视频验证卡顿？**
A: 降低视频分辨率或选择更小的模型

**Q: 标注框位置不准确？**
A: 使用标注模式手动调整，或调整 NMS 阈值
