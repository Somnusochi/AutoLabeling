# 推理性能基准

## Windows 11 + RTX 3080 10GB (CUDA)

13 张猫图（Pexels，混合分辨率），各 4 轮，自动长边 800px，`max_new_tokens=512`。

| 模式 | 图像尺寸 | 冷启动 (R1首张) | 各轮详情 (R2/R3/R4) | 稳定平均 (热启动) | 峰值显存 |
|------|---------|-----------|--------------------|-----------------|---------|
| VLM only | 大图 (800px) | 8.26s | 7.43s / 6.61s / 5.73s | **6.59s/张** | ~9.8 GB |
| VLM only | 缩略图 (256px) | 0.70s | 1.28s / 1.15s / 1.68s | **1.37s/张** | ~10.0 GB |
| VLM + SAM2 | 大图 (800px) | 1.27s | 1.86s / 1.85s / 1.86s | **1.86s/张** | ~9.0 GB |
| VLM + SAM2 | 缩略图 (256px) | 1.76s | 0.82s / 0.83s / 0.83s | **0.83s/张** | ~9.8 GB |

> *应用真正冷启动（服务首次启动后首个 API 调用）：**~0.70s**。模型权重经初次加载后进入 OS 文件缓存，后续加载远快于首次完全冷启动。
> *套件顺序执行（VLM-only → VLM+SAM2）；后执行的套件受益于已编译的 CUDA 计算图和稳定后的显存分配。
>
> **显存压力是 10 GB 显卡的主要瓶颈**：VLM-only 大图处理因显存峰值达 9.8 GB（占总量 96%）导致单张耗时波动极大（1.7–16.5s）。VLM+SAM2 大图在显存稳定于 ~9.0 GB 后性能极为一致（1.85–1.86s，标准差 < 0.01s）。
>
> **最快热启动单张耗时**：VLM-only 缩略图 ~0.58s，VLM-only 大图 ~1.74s，VLM+SAM2 缩略图 ~0.76s，VLM+SAM2 大图 ~1.27s。完全预热后 SAM2 开销极小（约 +0.03–0.06s/张）。
>
> **建议**：强烈推荐 12 GB+ 显卡以保证稳定生产推理。10 GB 可使用，但在持续大图负载下存在显存分配器颠簸问题。

| 模式 | 图像尺寸 | 冷启动 (R1首张) | 各轮详情 (R2/R3/R4) | 稳定平均 (热启动) | 系统内存开销 |

## Linux + RTX 4080 16GB (CUDA)

13 张猫图，各 4 轮，自动长边 1024px (大图) / 256px (缩略图)。

| 模式 | 图像尺寸 | 冷启动 (R1首张) | 各轮详情 (R2/R3/R4) | 稳定平均 (热启动) | 显存/内存开销 |
|------|---------|-----------|------------------|-----------------|-------------|
| VLM only | 大图 (1024px) | 2.14s | 2.76s / 2.75s / 2.75s | **2.75s/张** | ~7.7GB (峰值) |
| VLM only | 缩略图 (256px) | 12.37s* | 0.64s / 0.59s / 0.58s | **0.60s/张** | ~7.7GB (峰值) |
| VLM + SAM2 | 大图 (1024px) | 2.62s | 3.69s / 3.74s / 3.69s | **3.71s/张** | ~8.2GB (峰值) |
| VLM + SAM2 | 缩略图 (256px) | 2.49s | 0.96s / 0.92s / 0.94s | **0.94s/张** | ~8.2GB (峰值) |

> *完全冷启动 (12.37s) 包含模型权重首次载入显存与 PyTorch 计算图的初次构建。其余"首张"为同进程下模型预热后的切换耗时。
> **注**：在高频大图推理中，VLM+SAM2 平稳后的理论耗时极度稳定，基本平滑锁定在 3.71s。

## 配置说明

- **长边上限**：根据 GPU 显存自动选择（<12GB → 800px，12–16GB → 1024px，≥16GB → 1333px；MPS → 1024px）
- **GPU 内存管理**：策略模式（`gpu_memory.py`）— CUDA `expandable_segments:True` / MPS `synchronize`+`empty_cache`+`gc`
- **SAM2 模型**：`facebook/sam2.1-hiera-base-plus`
